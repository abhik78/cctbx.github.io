cctbx.maptbx.real_space_refinement module¶

class cctbx.maptbx.real_space_refinement.lbfgs(basic_map, sites_cart, delta_h=1.0)¶

Bases: object

If you run the lbfgs with a non-crystallographic basic_map, and it throws an out-of-bounds error, then change the out_of_bounds handle in the basic_map to:

my_basic_map.set_out_of_bounds_handle(maptbx.out_of_bounds_clamp(0).as_handle())
